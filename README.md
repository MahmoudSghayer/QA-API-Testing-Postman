# API Testing – Fake Store (Postman)

## Project Overview
This project demonstrates manual API testing using Postman on the Fake Store public API.
It covers positive and negative test scenarios with response validations.

## Tools Used
- Postman

## Tested Endpoints
- GET /products
- GET /products/{id} (invalid ID)

## Testing Scope
- Status code validation
- Response body validation
- Field-level validation
- Negative API testing
